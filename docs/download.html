Ссылки для загрузки

К загрузке предлагаются несжатые модули. О том, как сжать их и о возможных последствиях читайте здесь.

Все версии модулей (7zSD) для инсталляторов использует MSVCRT.dll (MSVCRT.dll - Microsoft Visual C runtime library). Т.е. часть функций/подпрограмм языка вынесена в отдельный файл MSVCRT.dll, что позволило значительно уменьшить размер модулей. Этот файл есть во всех системам поддерживаемых модулем. Старые версии модулей (7zS), для которых не требовался этот файл, более не поддерживаются.

Большинство файлов, предложенных для скачивания в таблице этой страницы, упаковано в 7z архивы, поэтому, позаботьтесь, чтобы у Вас было чем их распаковать.

Другие версии файлов можно найти здесь.

(7z SFX архив)

881 кб

Набор для работы с 7z SFX архивами. Состав: 12 модифицированных модулей (2 из них с добавленными диалогами и баннером, один из который использовался для создания этого архива), 7z SFX Archive Splitter, 2 справочных файла (русский и английский).
